# Working-Hangman-With-Speech-For-a-Semi-Broken-Keyboard
Assuming your keyboard's keys are worn off, this project provides a way to use speech recognition to play the game, with minimal clicks.  Special thanks to https://github.com/kiteco/python-youtube-code/tree/master/build-hangman-in-python and https://github.com/Lavakumar999/Python-Tkinter-FrameWork/tree/master/hangman for the code and images.
